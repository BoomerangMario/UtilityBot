THIS SOURCE CODE IS FOR DISCORD.PY 2.0, Python 3.8 or higher is required.

=======================================

SETUP GUIDE:
1. Download or fork this Source code and create an application in discord.com/developers and copy your bot token. Make sure to turn on the Message Content intent and the Members intent.
2. At the bottom with client.run("BOTTOKENHERE"), replace BOTTOKENHERE with your bot token.
3. Towards the top at client = commands.Bot(command_prefix=commands.when_mentioned_or('c;')), replace c; with your prefix
4. Run the Bot.
